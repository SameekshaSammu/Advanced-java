# Advanced Java Programs

This repository contains a collection of Advanced Java programs demonstrating concepts like Swing, AWT, Collections, and String handling.

---

## List of Programs

### Swing & GUI Programs

* ButtonExample.java – Demonstrates button creation in Swing
* ImageButtonDemo.java – Button with image icon
* SwingExample.java – Basic GUI using Swing
* TabbedColorDemo.java – Tabbed pane with color display
* TabbedPaneExample.java – Demonstrates tabbed pane

### Collection Framework

* CountryCapitalList.java – Stores country-capital pairs
* JListExample.java – Demonstrates JList component
* ListInterfaceDemo.java – Uses List interface methods

### String Handling

* StringBuildingFunctions.java – StringBuilder methods
* StringHandlingDemo.java – String operations

### Performance

* PerformanceTest.java – Performance comparison program

### Additional Programs

* p1.java to p10.java – Practice / lab programs

---

## Technologies Used

* Java
* Swing
* AWT
* Collections Framework

---

## How to Run

1. Open the project in Eclipse / IntelliJ
2. Compile the program
3. Run any `.java` file
